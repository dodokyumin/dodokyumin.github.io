.stream();

list를 Stream으로 데이터의 흐름.

흐름을 타고 product들을 가져오는데 filter를 통해 거른다.

```java
    public Product getProductById(String id) {
    	return products.stream()
    			.filter((product) -> product.getId().equals(id))//filter로 조건 걸기
    			.findFirst() //첫번째
    			.get(); //얻어
    }
```
